## 首页颜色变化

修改文件 : `/source/css/matery.css`, 搜索 `rainbow` 来关闭颜色变换：
```css
@-webkit-keyframes rainbow {
   /* 动态切换背景颜色.即滤镜颜色，不想要可以全部注释，或者换成你喜欢的颜色 */
}

@keyframes rainbow {
    /* 动态切换背景颜色.，不想要可以全部注释，或者换成你喜欢的颜色 */
}
```
